You must integrate Afrocat operational digital assets into the portal.

These assets already exist outside the system and must be supported.

ASSET 1 — PLAYER PHOTO STORAGE

Player photos originate from Google Drive folder.

Create configuration table:

SystemAssetConfig {
 id String @id
 playerPhotoDriveFolderId String
 reportDriveFolderId String
}

Admin settings page must allow updating folder IDs.

Player photoUrl logic:
If photo missing locally,
system can reference Drive link.

ASSET 2 — PLAYER REPORT STORAGE

Performance reports are exported as PDF.

Reports must support:

- Auto-save location reference
- Download history
- Report metadata tracking.

Add:

PlayerReport {
 id
 playerId
 reportType
 pdfUrl
 generatedDate
}

ASSET 4 — LIVE MATCH STAT PANEL INTEGRATION
