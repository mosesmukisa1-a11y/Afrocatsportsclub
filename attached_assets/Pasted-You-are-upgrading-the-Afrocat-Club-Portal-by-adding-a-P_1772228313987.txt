You are upgrading the Afrocat Club Portal by adding a PROFESSIONAL PLAYER CONTRACT MANAGEMENT MODULE integrated directly inside Player Profiles.

GOAL
Every player profile must support:
- Contract creation
- Contract upload (PDF/DOC)
- Contract duration tracking
- Renewal alerts
- Financial obligations tracking
- Transport obligations tracking
- Contract approval workflow

DATABASE (PRISMA)

Create new model:

PlayerContract {
  id                String @id @default(cuid())
  tenantId          String
  playerId          String
  contractType      ContractType
  startDate         DateTime
  endDate           DateTime
  signOnFee         Float?
  weeklyTransport   Float?
  salaryAmount      Float?
  obligations       String?
  contractFileUrl   String?
  status            ContractStatus
  approvedByUserId  String?
  createdAt         DateTime @default(now())

  player Player @relation(fields:[playerId], references:[id])

  @@index([tenantId])
}

enum ContractType {
  PERMANENT
  SEASONAL
  TRIAL
  YOUTH
}

enum ContractStatus {
  DRAFT
  ACTIVE
  EXPIRED
  TERMINATED
}
