========================================================

SkillType:
- BLOCK (or ATTACK outcomeDetail)

RECOMMENDED:

- If tracking attacker: ATTACK success option “TOOL_BLOCK/BLOCK_OUT” = outcome="PLUS"

6) POINTS AUTO-UPDATE (RECOMMENDED)
========================================================

Add point attribution for each event:

Add MatchEvent field:
- pointWonByTeamId (nullable)

Rule of thumb:
- Any “MINUS” that results in immediate fault can set pointWonByTeamId=opponentTeamId
- Any “PLUS” kill/ace can set pointWonByTeamId=playerTeamId

Minimum reliable point triggers:
- SERVE ACE => point to server team
- SERVE NET/OUT => point to receiving team
- ATTACK KILL => point to attacker team
- ATTACK OUT/NET/BLOCKED => point to defender team
- BLOCK STUFF => point to blocker team
- RECEIVE 0_ERROR that ends rally => point to opponent team

When pointWonByTeamId is set:
- increment match.homePoints or match.awayPoints

Also provide Manual Buttons:
- “+1 Home”
- “+1 Away”
- “Undo Last Point”
for cases where rally outcome wasn’t captured.
